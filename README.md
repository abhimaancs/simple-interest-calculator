\# GitHub Final Project – Simple Interest Calculator



\## Project Description

This project is a simple Bash-based calculator that computes \*\*Simple Interest\*\*

based on user input. It is created as part of the GitHub Final Project assignment.



\## Formula Used

Simple Interest (SI) is calculated using the formula:



SI = (Principal × Rate × Time) / 100



\## How to Use the Script

1\. Run the `simple-interest.sh` script.

2\. Enter the following inputs when prompted:

&nbsp;  - Principal amount

&nbsp;  - Rate of interest

&nbsp;  - Time period (in years)

3\. The script will display the calculated simple interest.
